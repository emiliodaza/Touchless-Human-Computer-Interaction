# Multimodal-Human-Computer-Interaction-HCI-system
This project explores a multimodal HCI system that enables computer interaction through eye tracking, voice commands and hand gestures, either independently or in combination. It functions as an intent-mediated input layer designed for accessibility and natural interaction.

## 🚀 Installation & Quick Start (Offline Voice Access)

This project runs **fully offline** and is designed for **assistive computer control** using voice, light hand gestures, and minimal movement.

---

### 🧩 System Requirements

- **OS:** Windows / Linux / macOS  
- **Python:** 3.9 – 3.11  
- **Microphone:** Required  
- **Webcam:** Optional (for hand gestures)

---

### 📦 Required Python Libraries

Install dependencies using `pip`:

```bash
pip install pyautogui
pip install speechrecognition
pip install pocketsphinx
pip install opencv-python
pip install mediapipe
pip install numpy
